== Description ==

This plugin will help you to prevent auto comment spam submittions by clawlers and bots. Requires PHP5 and GD extention installed.

== Installation ==

1. Upload `wp_hookman_captcha` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
